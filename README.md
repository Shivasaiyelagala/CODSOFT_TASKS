# CODSOFT Data Analytics Internship

## Task 1: Data Cleaning & Preprocessing

### Objective
The objective of this project is to clean and preprocess the Titanic dataset using Python and Pandas.

### Dataset
Titanic Dataset

### Tools Used
- Python
- Pandas

### Steps Performed
- Loaded the dataset
- Inspected the dataset
- Checked missing values
- Filled missing values in the Age column
- Filled missing values in the Embarked column
- Removed the Cabin column because it had many missing values
- Checked for duplicate records
- Saved the cleaned dataset as `cleaned_titanic.csv`

### Output
The cleaned dataset was successfully saved as:
```
cleaned_titanic.csv
```
